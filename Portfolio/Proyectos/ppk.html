Proyecto: Programa para Determinar el PPK del Espesor de Aluminio

A lo largo de mi experiencia en la industria, identificando la importancia del control de calidad y la necesidad de herramientas eficientes para los técnicos, surgió la idea de desarrollar un proyecto práctico mientras comenzaba a estudiar Desarrollo Web. Este programa está diseñado para simplificar el proceso de medición del espesor del aluminio, permitiendo a los técnicos ingresar los valores que genera el sistema oficial de medición y convirtiéndolos automáticamente a milésimas (por ejemplo, al ingresar "580", se transforma en "0,0058").

Una vez completadas las seis mediciones que conforman cada control, el sistema calcula automáticamente la media y realiza los cálculos necesarios para determinar el valor del PPK, un índice esencial en la industria para evaluar la capacidad del proceso de producir productos de calidad.

Los resultados del índice PPK se interpretan de la siguiente manera:

  • Valor menor a 1 - Proceso No aceptable
  • Valor mayor a 1 - Proceso aceptable
  • Valor mayor a 1,33 - Productos dentro de las especificaciones de calidad, con resultados consistentes y cercanos a la media.

Este proyecto está diseñado para trabajar con tres formatos de productos diferentes y dos medidas de calidad, lo que permite al usuario seleccionar la opción deseada mediante dos botones. Según la elección, el sistema mostrará los valores de referencia correspondientes.
